Welcome to SpringBoot Day 1
SpringBoot is Spring module from Spring Framework.
SpringBoot is used for RAD(Rapid application Development).
RAD is modified waterfall model(SLDC) which focuses on developing software in short span of time.

Phase of RAD are as follows
1.Business Model
2.Data Model
3.Process Model
4.Application Generation
5.Testing & turnover
We can change the default port no i.e., in appliation.properties file with server.port=XXXX(any four digits)
